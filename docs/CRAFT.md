# 项目描述

此应用要做的是一个板材编辑应用。通过在 3D 页面和 2D 页面上编辑板材的曲线、形状、拼接方式，来实现对某个 creative assets 的编辑。这个应用很像是一个 CAD 软件。但是我们不会做这么重，因为这个是一个技术探索项目。

这个项目会分为两个页面：

- `editor`：编辑器，用户可以在这个页面上编辑板材的曲线，以及拼接方式。我们会在两块板的黏合处增加吸附功能，然后自动生成拼接处的曲线。例如榫卯结构，或者外挂+螺丝结构。这个前期可以不要让用户自己设计拼接方式，我们先内置一些常见的拼接方式，用户可以选择使用。编辑器有 2D 和 3D 两个视图。2D 是平面视图（SVG），3D 是基于 React + Three.js 的视图。用户在 2D 视图上编辑曲线，3D 视图会实时反映用户的编辑结果。用户也可以在 3D 视图上直接编辑曲线，2D 视图也会实时反映用户的编辑结果。在编辑视图的时候可以做简单的 CSG 布尔运算，来帮助用户编辑。
- `generator`：编辑器的产物是一种 pattern。具体是 json 还是 yaml 我们可以商量这来。用户在 generator 页面上可以上传他们在 editor 页面上编辑好的 pattern，然后生成对应的 SVG 文件。

板材可以理解为一块木板，用户编辑后，会把 svg 图发给激光切割机，来切割出对应的板材。这个项目的目标是让用户能够通过编辑器来设计他们想要的板材，然后通过生成器来生成对应的 SVG 文件，来实现他们的创意。

这个项目有两种目标人群：

- 具有审美和动手能力的创意工作者：可以在编辑器中通过复杂的曲线编辑和拼接方式设计出独特的板材，来实现他们的创意。
- 没有太多建筑设计经验的普通用户：可以通过编辑器中的预设拼接方式和简单的曲线编辑，来设计出他们想要的板材。或者可以直接使用 generator 来生成对应的 SVG 文件，来实现他们的创意。

## 项目技术选型

- React, TypeScript, Vite, Bun, React Three Fiber, Tailwind CSS, Shadcn UI, React-BVH-CSG

## 项目结构

项目结构如下：

- `packages/core`: 包含主要的逻辑和算法。
- `packages/protocol`: 包含通用的协议，可以在编辑器和生成器中共享。
- `apps/website`: 包含两个主要的应用：
  - `editor`: 编辑器应用。
  - `generator`: 生成器应用。
